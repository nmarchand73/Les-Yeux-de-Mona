# Instructions de téléchargement manuel des images

## Méthode principale : Wikimedia Commons

**Toutes les images sont disponibles sur Wikimedia Commons**, ce qui simplifie grandement le processus de téléchargement.

---

## Méthode rapide (recommandée) - Utilisation de la recherche Wikimedia Commons

### Étapes générales pour toutes les œuvres :

1. **Allez sur** : https://commons.wikimedia.org/w/index.php?search=&title=Special%3AMediaSearch&type=image
2. **Dans la barre de recherche**, tapez le nom de l'œuvre et de l'artiste (ex: "Thomas Cole The Oxbow")
3. **Cliquez sur l'image** dans les résultats de recherche pour l'agrandir
4. **Cliquez sur "Download"** (en haut à droite de l'image)
5. **Choisissez "Original file"** ou la plus grande taille disponible
6. **Enregistrez** avec le nom exact indiqué ci-dessous dans le dossier `images/hudson_river/`

**Astuce** : Vous pouvez aussi utiliser la recherche avancée en tapant : `"[Nom de l'artiste]" "[Titre de l'œuvre]"` pour des résultats plus précis.

## Instructions détaillées pour chaque téléchargement

Pour chaque œuvre, suivez ces étapes :

1. **Allez sur** : https://commons.wikimedia.org/w/index.php?search=&title=Special%3AMediaSearch&type=image
2. **Dans la barre de recherche**, tapez les termes de recherche indiqués ci-dessus (ex: "Thomas Cole The Oxbow")
3. **Cliquez sur l'image** dans les résultats de recherche pour l'agrandir
4. **Cliquez sur le bouton "Download"** (icône de flèche vers le bas, en haut à droite)
5. **Sélectionnez "Original file"** dans le menu déroulant
6. **Enregistrez le fichier** dans le dossier `images/hudson_river/`
7. **Renommez le fichier** avec le nom exact indiqué (si nécessaire)

**Note** : Les fichiers téléchargés depuis Wikimedia Commons peuvent avoir des noms longs. Assurez-vous de les renommer exactement comme indiqué ci-dessus.

## Astuce : Téléchargement en lot

Si vous souhaitez télécharger plusieurs images rapidement :

1. Ouvrez la page de recherche : https://commons.wikimedia.org/w/index.php?search=&title=Special%3AMediaSearch&type=image
2. Pour chaque œuvre, recherchez l'image et cliquez sur "Download" → "Original file"
3. Les fichiers se téléchargeront dans votre dossier de téléchargements par défaut

## Alternative : Recherche manuelle

Si la recherche ne fonctionne pas, vous pouvez rechercher manuellement :

1. Allez sur https://commons.wikimedia.org/w/index.php?search=&title=Special%3AMediaSearch&type=image
2. Dans la barre de recherche, tapez : `"[Nom de l'artiste]" "[Titre de l'œuvre]"`
   - Exemple : `"Thomas Cole" "The Oxbow"`
3. Cliquez sur le résultat correspondant dans les résultats de recherche
4. Suivez les étapes de téléchargement ci-dessus

## Utilisation de la nouvelle interface de recherche Wikimedia Commons

La nouvelle interface de recherche de Wikimedia Commons (https://commons.wikimedia.org/w/index.php?search=&title=Special%3AMediaSearch&type=image) offre plusieurs avantages :

- **Recherche visuelle** : Les résultats s'affichent sous forme de grille d'images
- **Filtres avancés** : Vous pouvez filtrer par type de média, licence, etc.
- **Prévisualisation** : Passez la souris sur une image pour voir un aperçu
- **Téléchargement rapide** : Accès direct au bouton de téléchargement depuis les résultats
